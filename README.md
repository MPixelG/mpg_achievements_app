# MPG-Pixelverse

A 3D game using Flutter, Flame and thermion. Setting and storyline MPG. Contribute to the community at your school and become a change-maker by helping with real-life social, environmental or financial challenges and receive rewards. Additionally, learn OOP with challenges and tasks in various scenarios.
Made by pupils for pupils!
